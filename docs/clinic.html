Clinic API

Each call to the Order API will reqiure an associated Clinic. The Clinic API can be used to associate a new Clinic with your authorization key, or validate the status of an existing Clinic.

Create a Clinic

Associate a new Clinic with your authorization key.

Parameters
authkey
required
string (64)
Your 64-character API key
name
required
string (128)
The name of the Clinic
email
required
string (128)
An email address associated with the Clinic
website
required
string (128)
A website associated with the Clinic
phone
optional
string (16)
An phone number associated with the Clinic

Format: (555) 555-5555
callback_url
optional
string (128)
An 'update notification' callback URL, which is required to be HTTPS URL.
address_line1
required
string (64)
Address Line 1
address_line2
optional
string (16)
Address Line 2
address_city
required
string (32)
Address City
address_province
required
string (2)
Address Province Code

Format: MB
address_postal
required
string (7)
Address Postal Code

Format: A1B 2C3
address_country
required
string (2)
Address Country Code

Format: CA
supplier_alcon
optional
string (12)
Clinic's Alcon account number

Note: Optional, but will be required to order Alcon product
supplier_bausch
optional
string (12)
Clinic's Bausch + Lomb account number

Note: Optional, but will be required to order Bausch + Lomb product
supplier_cooper
optional
string (12)
Clinic's CooperVision account number

Note: Optional, but will be required to order CooperVision product
supplier_johnson
optional
string (12)
Clinic's Johnson & Johnson account number

Note: Optional, but will be required to order Johnson & Johnson product
Success Response
id
string
The resulting Location ID to be used in Order calls
location_apiKey
string (64)
The location's 64-character API key
