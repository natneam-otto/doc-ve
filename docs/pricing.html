Pricing API

The Pricing API allows you to fetch information about the available product pricings available through Otto.

Fetch Pricing

Useful for getting prices for a given clinic. It can be filtered by product id and enabled status.

Parameters
location_apiKey
required
string
Locaiton's 64-character API key
product_id
optional
id
A specific Otto product id. Useful for getting a specific product price.
enabled_products
optional
boolean
When set to true enabled_products filters the result to only show products which are enabled for the given location.
Success Response
data
array
An array of pricemap objects, with parameters of those objects found below:
pricemap_id
int
The id for the given price map object. Useful for updating prices (Coming soon)
pricemap_productID
int
An Otto Product Id
pricemap_price
double
Pricing for the given product at the given location
pricemap_enabled
int
An int value representing the enabled status of the given product. (1: enabled, 0: disabled)
pricemap_supplierID
int
The supplier currently set for the given product represented by a Otto Supplier Id.
pricemap_taxable
int
The tax status for a given product represented by a boolean int value (1: taxable, 0: non taxable)
supplyprice_id
int
The Id of the custom volume/supply pricing
supplyprice_value
float
The price for the given product at the given volume quantity
supplyprice_quantity
int
The quantity required for the given discounted volume price
