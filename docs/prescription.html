Create/Update Prescription

Useful for creating and updating prescription.

Parameters
location_apiKey required
string
Your 64-character API key
prescription_expirationDate optional
string
Prescription expiration date, in mm/dd/yyyy format.
refill_reminderDate optional
string
Scheduled date for sending refill reminder to patient, in mm/dd/yyyy format.
prescription_doctor optional
string
Prescribing Doctor
patient_id required when creating new prescription
integer
Patient ID
prescription_id required when updating prescription
integer
Prescription ID
prescriptions
required
array
An array of at most two line items to include in the prescription, with parameters of those objects found below:
product_id
required
int
Otto Product ID.
The ID of the supplier of the product.
product_eye
optional *
string
  • OD: Right Eye
  • OS: Left Eye
* Required when the prescription is for contact lenses
upc
optional *
string
A upc to identify the product to be ordered.

* Required when product_sph is not provided.
product_sph
optional *
string
SPH value of the product to be ordered.

* Required when product_upc is not provided.
product_cyl
optional *
string
CYL value of the product to be ordered.

* Required when product_upc is not provided and product_id requires a CYL value.
product_axis
optional *
string
AXIS value of the product to be ordered.

* Required when product_upc is not provided and product_id requires an AXIS value.
product_bc
optional *
string
BC value of the product to be ordered.

* Required when product_upc is not provided.
product_dia
string
DIA value of the product to be ordered.

* Required when product_upc is not provided.
product_add
string
ADD value of the product to be ordered.

* Required when product_upc is not provided and product_id requires an ADD value.
product_dn
string
D/N value of the product to be ordered.

* Required when product_upc is not provided and product_id requires a D/N value.
product_colours
string
COLOUR value of the product to be ordered.

* Required when product_upc is not provided and product_id requires a COLOUR value.
Success Response
prescription_id
id
A resulting prescription_id
