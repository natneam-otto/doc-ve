Patient API

The Patient API allows you to fetch information about a patient.

Fetch Patient

Useful for fetching details about a patient.

Parameters
location_apiKey
required
string
Your 64-character API key
patient_id
required
integer
A patient ID which uniquely identifies a patient
Success Response
data
object
A Patient object, parameters of the object found below:
patient_firstName
string
Patient's first name
patient_lastName
string
Patient's last name
patient_mobile
string
Patient's phone number
patient_email
string
Patient's email address
patient_addressLine1
string
Patient's address line 1
patient_addressCity
string
Patient's city
patient_addressProvince
string
Patient's province
patient_addressPostalCode
string
Patient's postal code
patient_country
string
Patient's country
prescriptions
array
An array of patient's prescription ID
